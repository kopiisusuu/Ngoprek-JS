Function (Fungsi)

Function adalah blok kode yang dirancang untuk melakukan tugas tertentu. Function dapat dipanggil kapan saja dan membantu dalam mengorganisir kode menjadi bagian-bagian yang dapat digunakan kembali.

Jenis-jenis Function

  • Function Declaration - Fungsi yang dideklarasikan dengan kata kunci function
  • Function Expression - Fungsi yang disimpan dalam variabel
  • Arrow Function - Sintaks fungsi yang lebih ringkas (diperkenalkan di ES6)

Contoh Function

// Function Declaration
function sapa(nama) {
  return "Halo, " + nama + "!";
}

// Function Expression
const tambah = function(a, b) {
  return a + b;
};

// Arrow Function
const kali = (a, b) => a * b;

// Memanggil fungsi
const hasilSapa = sapa("Budi");
const hasilTambah = tambah(5, 3);
const hasilKali = kali(4, 6);

Method

Method adalah fungsi yang terkait dengan objek. Dalam JavaScript, method adalah properti dari objek yang berisi definisi fungsi.

Jenis-jenis Method

  • Built-in Methods - Method yang sudah disediakan oleh JavaScript
  • Custom Methods - Method yang dibuat sendiri oleh developer

Contoh Method

// Built-in Methods
const pesan = "Hello World";
const pesanKapital = pesan.toUpperCase(); // toUpperCase() adalah method

// Custom Methods dalam objek
const kalkulator = {
  nilai: 0,
  tambah: function(angka) {
    this.nilai += angka;
    return this.nilai;
  },
  kurang: function(angka) {
    this.nilai -= angka;
    return this.nilai;
  },
  reset: function() {
    this.nilai = 0;
    return this.nilai;
  }
};

Array

Array adalah struktur data yang digunakan untuk menyimpan kumpulan nilai dalam satu variabel. Setiap nilai dalam array disebut elemen, dan setiap elemen memiliki posisi yang disebut index (dimulai dari 0).

Method Array yang Umum Digunakan

  • push() - Menambahkan elemen di akhir array
  • pop() - Menghapus elemen terakhir array
  • shift() - Menghapus elemen pertama array
  • unshift() - Menambahkan elemen di awal array
  • map() - Membuat array baru dengan hasil pemanggilan fungsi pada setiap elemen
  • filter() - Membuat array baru dengan elemen yang lolos tes
  • forEach() - Menjalankan fungsi untuk setiap elemen array

Contoh Array

// Membuat array
const buah = ["Apel", "Mangga", "Jeruk"];

// Mengakses elemen array
const buahPertama = buah[0]; // "Apel"

// Mengubah elemen array
buah[1] = "Pisang"; // ["Apel", "Pisang", "Jeruk"]

// Menambahkan elemen
buah.push("Anggur"); // ["Apel", "Pisang", "Jeruk", "Anggur"]

// Menghapus elemen terakhir
buah.pop(); // ["Apel", "Pisang", "Jeruk"]

// Method map untuk transformasi array
const bilangan = [1, 2, 3, 4];
const dikaliDua = bilangan.map(function(angka) {
  return angka * 2;
}); // [2, 4, 6, 8]
